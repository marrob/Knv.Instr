NI-DCPower .NET Class Library 21.3 Readme

The .NET class library for NI-DCPower provides an IVI class-compliant .NET API interface to the underlying NI-DCPower instrument driver API.

Note    The NI-DCPower .NET Class Library Help is no longer integrated into the Visual Studio Help Viewer. Select Start»All Programs»National Instruments»NI-DCPower»NI-DCPower Documentation to view the help.

Installing the NI-DCPower .NET Class Library

To use NI-DCPower .NET class library support, your computer must have the following:

  • Microsoft operating system:
    • Windows 10
    • Windows Server 20191
    • Windows Server 2016
  • Microsoft .NET Framework 4.0 or later

1NI software installs VC2015 Runtime and .NET 4.6.2.

Installation Notes

  • You must install the NI-DCPower 21.3 (minimum) instrument driver in order to use the NI-DCPower .NET Class Library. Refer to NI Drivers and Updates on ni.com and enter DCPower into the search field to download the latest version of the NI-DCPower instrument driver.
  • If Windows Update is enabled, the installation process might hang if Windows Update interferes with the installation of Microsoft Visual C++ 2015 Run-Time. Visit ni.com/info and enter the Info Code exjq43 for more information and steps to resolve this issue.
  • This product supports, and was tested with, the latest Visual Studio service pack and the latest operating system service pack that was available at the time this product was released. NI strongly recommends using NI-DCPower with the latest Visual Studio service pack and the latest operating system service pack.
  • If you want to upgrade your operating system from one major version to another, NI recommends first uninstalling all NI software, including application software and drivers. This is typically not necessary when installing an operating system service pack.

Installation Instructions

  1. Download the self-extracting .exe file that contains the NI-DCPower .NET Class Library.
  2. Double-click the .exe, verify the install location, then select Unzip to copy the contents of the file to the specified location.
  3. If selected, setup.exe will launch automatically. Otherwise, select setup.exe to launch the NI-DCPower .NET Class Library installer.
  4. Follow the instructions that appear on the screen.

Known Issues in NI-DCPower .NET Class Library 21.3

The following are known issues in NI-DCPower .NET Class Library 21.3. For a list of known issues in the underlying driver, refer to NI-DCPower Known Issues.

  • The NI-DCPower examples may produce a warning - "Warning: Found conflicts between different versions of the same dependent assembly". You can safely ignore this warning.
  • When the NI-DCPower .NET class library is installed, some assemblies specific to previous versions of NI-DCPower could be deleted from the GAC. The affected assemblies are NationalInstruments.ModularInstruments.NIDCPower.dll, NationalInstruments.ModularInstruments.Common.dll, and NationalInstruments.ModularInstruments.ModularInstrumentsSystem.dll. Any application which refers to specific versions of these assemblies will not compile once later version of the NI-DCPower .NET class library is installed.
